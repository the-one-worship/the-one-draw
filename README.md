# The One Linked

## Development

### Setup

Install [pnpm](https://pnpm.io/) for manage package
and make sure to install the dependencies:

```bash
pnpm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm run dev
```
